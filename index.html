What is Facil?

Facil is a static website generator written in Go. The project can be found on Github.

Why write it?

I have an interest in content management, and I’m still trying to improve my Go programming skills.

Is it any good?

Well it compiled this static website. It’s ok I think, but decide for yourself

Tell me more

Facil is written in Go. The project is MIT licensed and fully open sourced. You can find it here on github.

It’s a command line tool that scaffolds websites with text/markdown content files using theme templates. You just need to edit the text/markdown files and then build the site. Building will output a static html website that you can upload to just about anywhere. Facil supports HTTPS and Pretty URLs.

Facil is a work in progress. Some of the code is very ugly by Go standards and there are currently no tests. It’s very fast to build websites to static html, but I haven’t explored concurrency. The default theme that ships with it is nothing more than a testbed for Facil functionality. These are all things I hope to improve.
